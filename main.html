    FAQ

    What is AskTrump?
    AskTrump is the web interface for TrumpSim, a “What would Trump do?” style advisory engine. You ask a question and get a single, bold response in the spirit of a negotiator and leader.
    Do I need an account?
    No. The site runs in your browser and talks to the TrumpSim server. No sign-up or login is required.
    Is this legal or financial advice?
    No. All responses are for entertainment and inspiration only. They are not legal, tax, regulatory, or investment advice.
    Where does the server run?
    By default the page expects TrumpSim at http://localhost:2847. Run java TrumpSim from the TrumpSim folder to start the server, then open this page or go to that URL.
    Can I change the server URL?
    Yes. If you have a custom API base URL, change the API_BASE variable in the script section of this page (or in a separate app.js if you split it out).
